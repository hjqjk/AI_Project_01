# AI_Project_01

一个带有日历视图的代办事项管理单页应用。页面使用原生 HTML/CSS/JS 构建，无需任何依赖，直接打开 `index.html` 即可体验。

## 功能概览
- 创建任务：录入标题、描述、截止日期与优先级。
- 日历视图：按月展示日期，并用彩色圆点标记对应日期的任务密度。
- 列表视图：按日期和优先级筛选、标记完成、删除任务。
- 示例与状态：可一键填充示例数据，支持本地存储，刷新后保留任务。

## 使用方式
1. 直接在浏览器中打开 `index.html`。
2. 在左侧表单中填写任务信息并提交；右侧日历会同步显示任务分布。
3. 点击日历日期可快速筛选当日任务，或使用顶部“回到今天”按钮跳转。
4. 若需要清空数据，使用“清空全部”按钮。

> 如需重置示例或删除本地数据，可在浏览器开发者工具中清空 `localStorage` 的 `tasks` 键。
